#### 1. 需求概述

> 页面随机N个颜色各异的气球,坐标位置随机,点击`Start`按钮,所有气球随机朝各个方向匀速运动,遇到窗口边框时按反方向运动,点击`Pause`安丘,所有小球停止运动.

#### 2. 目标知识点
- JavaScript面向对象
- DOM操作
- 定时器
